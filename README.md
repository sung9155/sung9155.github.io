# 단타로 살 수 있을까 · 현실 시뮬레이터

"취업 대신 주식 단타로 살겠다"는 선택을 **숫자와 그래프로 검증**해보는 한 장짜리 웹페이지입니다.
승률·손익비를 아무리 좋게 넣어도 거래비용과 변동성이 어떻게 작동하는지, 그리고 생활비를
단타 수익으로만 충당할 때 계좌가 언제 0원이 되는지를 직접 슬라이더로 확인할 수 있습니다.

🔗 **라이브:** https://sung9155.github.io/

## 구성 (탭 3개)

| 탭 | 내용 |
|----|------|
| **① 적립 비교** | 취업 나이·저축률·연 수익률을 바꿔, 늦게 시작해도 기준선(25세·33%·4%)을 65세에 따라잡는지 비교 |
| **② 단타 vs 취업+적립** | 같은 시드로 1,000명이 5년간 단타한 결과 분포를, 취업해서 지수에 적립한 사람과 비교. 승률 50%·손익비 1.0이어도 왜 대부분 지는지 보여줌 |
| **③ 단타로 생존** | 무직 상태에서 생활비를 단타로만 충당할 때, 1,000명을 10년 돌려 파산 시점·파산 확률을 보여줌 |

## 모델 가정

- **거래당 위험** 시드의 1%(손절 −1%, 익절 손익비×1%), **왕복 거래비용** 0.2%(증권거래세·수수료·슬리피지)
- **②** 연 250거래일, 1,000명 몬테카를로. 취업+적립은 시드+월 적립액을 연 6% 지수에 월복리
- **③** 매달 21거래일 × 1일 10회 매매 후 생활비 차감, 잔고 0 이하면 파산. 1,000명 10년 시뮬레이션
- 1:1 손익비 기준 거래비용·변동성 손실(volatility drag) 때문에 **승률 약 60% 이상**이어야 겨우 지수를 따라잡음

## 기술

- 단일 `index.html` (의존성 없음)
- 차트: [Chart.js 4](https://www.chartjs.org/) (CDN)
- 폰트: Pretendard, Fraunces
- 시뮬레이션: 클라이언트 사이드 몬테카를로(브라우저에서 즉시 계산)

> 정보 제공·교육용이며 특정 투자 권유가 아닙니다. 단순화된 모델로 실제 수익률은 변동하며 손실이 발생할 수 있습니다.
